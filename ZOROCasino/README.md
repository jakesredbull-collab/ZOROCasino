# ZORO Casino

## Project Structure

This repository contains the full project structure for ZORO Casino including frontend, backend, and database configuration.

```
ZOROCasino/
│
├── frontend/
│   ├── public/
│   ├── src/
│   ├── package.json
│   └── README.md
│
├── backend/
│   ├── src/
│   ├── package.json
│   └── README.md
│
└── database/
    ├── migrations/
    ├── seeds/
    ├── config.json
    └── README.md
```

## Description

- **frontend/**: Contains all the front-end source code and assets for the ZORO Casino application.
- **backend/**: Contains the server-side application code.
- **database/**: Contains database configurations and scripts for migrations and seeding.
